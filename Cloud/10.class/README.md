# Azure Big Data와 Machine Learning 서비스

## Azure Big Data와 Machine Learning 서비스 개요
Azure는 대규모 데이터 처리(빅데이터)와 인공지능/머신러닝 개발을 위한 다양한 클라우드 서비스를 제공한다.  
빅데이터 기술은 방대한 데이터를 저장·처리·분석하여 가치를 추출하는 데 활용되며, 머신러닝은 데이터를 학습하여 예측/의사결정을 수행하는 기술이다.

---

## 1. 빅데이터 (4차산업혁명의 키워드)

### 빅데이터란?
- 기존 데이터베이스 관리도구로 데이터를 수집, 저장, 관리, 분석할 수 있는 역량을 넘어서는 기술
- 대량의 **정형 또는 비정형 데이터 집합**을 다루고, 이러한 데이터로부터 **가치를 추출**하고 **결과를 분석**하는 기술
- 빅데이터를 활용함으로써 다변화되고 개인화된 현대사회에서 구성원들에게 **개별 맞춤형 정보 제공 / 관리 / 분석** 가능
- 인간이 간과하기 쉬운 정보들을 제공하여 **판단 및 결정에 도움을 주는 도구**로 활용 가능

### 빅데이터의 활용 가치
- 개인 맞춤형 정보 제공
- 서비스 개선 및 최적화
- 데이터 기반 의사결정 지원
- 새로운 비즈니스 기회 발굴

---

## 2. 빅데이터 활용 사례

- 서울시 심야버스 노선도
- 신용평가모델
- 금융 신상품 개발  
  - 예: 카드사 고객맞춤 상품 출시
- 택배 이용률 분석

---

## 3. Azure 빅데이터 및 분석 서비스

### 1) Azure SQL Data Warehouse
- 대규모 병렬 처리(MPP, Massive Parallel Processing)를 활용
- 페타바이트 규모의 데이터에서 복잡한 쿼리를 신속하게 실행 가능한 기업형 관리 서비스
- 대규모 데이터 분석 및 데이터 웨어하우징에 적합

> 참고: 현재 Azure에서는 관련 기능이 Azure Synapse Analytics로 확장/통합되어 설명되는 경우가 많음

### 2) Azure HDInsight
- 하둡 파일 시스템(HDFS)을 기반으로 동작
- 방대한 양의 데이터를 더 쉽고 빠르며 비용 효율적으로 처리하기 위한 **관리형 오픈소스 분석 서비스**
- Hadoop, Spark, Hive 등 빅데이터 프레임워크 활용 가능

### 3) Azure Data Lake Analytics
- 빅데이터를 단순화하는 **주문형(On-demand) 분석 작업 서비스**
- 하드웨어를 직접 배포하고 튜닝하는 대신, 쿼리를 작성하여 데이터를 변환하고 중요한 통찰력을 추출 가능
- 인프라 관리 부담을 줄이고 분석 작업에 집중 가능

---

## 4. 머신러닝 (Machine Learning)

### 머신러닝이란?
- 데이터를 구문 분석하고 해당 데이터를 통해 학습한 후,
- 학습한 내용을 바탕으로 결정을 내리기 위해 적용하는 알고리즘

### 예시
- 주문형 음악 스트리밍 서비스  
  - 사용자의 선호도/청취 패턴을 학습하여 맞춤형 추천 제공

### 머신러닝의 핵심기술
- **표현(Representation)**  
  - 데이터를 모델이 이해할 수 있는 형태로 표현하는 기술
- **일반화(Generalization)**  
  - 주어진 데이터로 기능을 수행하고, 시간이 지남에 따라 그 기능이 점차 향상되는 것  
  - 학습한 내용을 새로운 데이터에도 잘 적용하는 능력

---

## 5. Azure 머신러닝 서비스 (Azure Machine Learning)

### 개요
Azure 머신러닝 서비스는 기계 학습 모델의 **개발, 학습, 테스트, 배포, 관리 및 추적**에 사용되는 클라우드 기반 환경을 제공한다.

### 주요 특징
- 머신러닝 모델 개발부터 운영까지 전체 라이프사이클 지원
- 코드 작성 없이도 사용할 수 있는 **시각적 작업 영역(Visual Workspace)** 제공
- **드래그 앤 드롭(끌어서 놓기)** 방식으로 기계 학습 솔루션 빌드, 테스트, 배포 가능
- 공동 작업 기반 환경 제공 (협업 가능)

---

## 요약

**빅데이터와 머신러닝의 기본 개념**과 함께, 이를 Azure에서 어떻게 지원하는지 학습  
빅데이터는 대량의 정형/비정형 데이터를 분석하여 가치와 통찰력을 얻는 기술이며, Azure는 **Azure SQL Data Warehouse, HDInsight, Data Lake Analytics** 등의 서비스를 통해 대규모 데이터 처리와 분석을 지원한다. 
또한 머신러닝은 데이터를 기반으로 학습하여 예측/판단을 수행하는 기술이며, Azure Machine Learning 서비스를 통해 모델의 개발부터 배포, 관리, 추적까지 효율적으로 수행할 수 있음을 이해했다.
